# Micronaut Icon Assets

`micronaut-sally.svg` is copied from the official Micronaut brand guidelines
logo download for Sally, the Micronaut mascot.

Use remains subject to the Micronaut brand guidelines.

`languages/` contains project-owned monochrome SVGs used for generic code
language and file-format tabs. These are intentionally simple line icons so
they inherit the surrounding tab color and avoid competing with code content.
